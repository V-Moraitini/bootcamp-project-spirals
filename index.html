Bootcamp 2021: Making spirals

Warning: flashing lights!

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). This Processing project demonstrates a comprehensive understanding of the use of classes, methods, variables and randomness. It was primarily inspired by the creation of spirals and how different variables can generate different results. The user can change the time it takes to create a new segment or "dot" of the spiral (AKA arrangement), the rate of change of the distance away from the centre of the canvas and the radius of the segments to produce aesthetic results. The initialisation button will clear the canvas for a new spiral to be made in random but colourations and the "L" key will aid in the understanding of how the "arrangement" variable changes the shape of the spiral by making indicative lines sprouting from the centre.

Check out the code on Github: https://radujianu.github.io/
